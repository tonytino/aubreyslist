- `[propagate]` Fixed `/api/*` returning the SSR 404 page in production builds (sign-in/out, the health route, and any Hono endpoint were dead once built — `pnpm dev` hid it). Root cause: the construct scaffold was missing `app/api.ts` — the entry file that makes `@tanstack/react-start-config` register the Vinxi "api" router that serves API routes in the build — and `tsr.__enableAPIRoutesGeneration` was unset, so the file-based API route was never generated.
